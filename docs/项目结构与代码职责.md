# 项目结构与代码职责

## 1. 分层目标

- `Core`：放“契约”，不放实现。
- `Data`：放“持久化实现”，只负责读写与存储。
- `Pages`：放页面行为逻辑（`MainPage` 分域 partial）。
- `Views`：放 UI 组合视图（XAML + code-behind）。
- `Models`：放领域模型。
- `Controls`：放可复用控件。

## 3. 代码文件职责

### 3.1 Core（契约）

- `Core/Services/IWorkspaceService.cs`：工作区文档浏览、导入、读取、最近访问的统一接口。
- `Core/Services/IDrawingPersistenceService.cs`：手写状态保存/读取接口。

### 3.2 Data（持久化实现）

- `Data/Persistence/WorkspaceStore.cs`：工作区索引 `index.json` 与 PDF 文件存储读写。
- `Data/Persistence/DrawingStateStore.cs`：每个 note 的墨迹状态 JSON 持久化。

### 3.3 Pages（页面逻辑）

- `Pages/MainPage.xaml`：主页面组合入口（挂载 Home/Editor/Settings/Drawer/Toast 视图）。
- `Pages/MainPage.xaml.cs`：主页面生命周期与编辑器初始化入口。
- `Pages/Documents/Import.cs`：本地文件/URL 导入与打开流程。
- `Pages/Events/ViewEvents.cs`：所有 UI 事件绑定集中管理。
- `Pages/Navigation/BackNavigation.cs`：返回键与层级关闭策略。

#### Editor 域

- `DrawingCore.cs`：Editor 共享字段、枚举、基础守卫。
- `InputMode.cs`：手写/触摸/阅读模式切换与模式面板逻辑。
- `FloatingPanels.cs`：浮层面板定位、弹出、布局更新。
- `Tools.cs`：画笔/橡皮/颜色/粗细/压感工具行为。
- `Thumbnails.cs`：缩略图窗口、缓存、叠加渲染。
- `InkLifecycle.cs`：笔迹提交、裁剪、图层与保存节流。
- `PdfSearch.cs`：PDF 搜索、翻页、缩放状态同步。
- `ToolReorder.cs`：工具栏拖拽重排。

#### Home 域

- `Home.cs`：主页筛选、排序、抽屉入口、主页交互。
- `Workspace.cs`：工作区列表渲染、标签页、文档打开/切换。

#### Settings 域

- `Settings.cs`：设置页状态、显示/主题/语言等配置逻辑。

#### Shared 域

- `Animations.cs`：公共动画（弹入弹出、过渡）。
- `Localization.cs`：多语言文本映射与格式化。
- `MicroInteractions.cs`：微交互反馈（按压动画等）。
- `ThemePalette.cs`：主题调色板统一出口。
- `ViewBridge.cs`：跨 View 元素桥接访问。

### 3.4 Views（界面）

- `Views/Home/HomePanelView.*`：主页 UI。
- `Views/Editor/EditorChromeView.*`：编辑器顶部工具条、图层/缩略图等 UI 容器。
- `Views/Settings/SettingsOverlayView.*`：设置弹层 UI。
- `Views/Shared/DrawerOverlayView.*`：侧边抽屉 UI。
- `Views/Shared/StatusToastView.*`：状态提示气泡 UI。

### 3.5 其他基础模块

- `Controls/DrawingCanvas.cs`：底层绘制控件，手势、笔迹采样、渲染。
- `Models/*`：绘制状态与工作区实体模型。
- `Helpers/*`：图标渲染、ImageButton 图标尺寸与着色辅助。

## 4. 本次清理动作

- 删除无实际代码的空目录：`AI`、`Cloud`、`Collaboration`、`Plugins`、`Security` 等。
- 删除未被业务使用的样板能力（AI/安全/同步/仓储样板实现与接口）。
- 删除未使用 NuGet 包：`Syncfusion.Maui.Toolkit`。
- 页面/视图层级改为直接按职责分域，移除无意义中间目录。
- `Pages` 分部文件移除 `MainPage.` 冗余前缀，统一为职责型命名。
